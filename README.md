# Insurance Dataset Analysis in R

## Project Overview
This project analyzes an insurance dataset using R to identify factors influencing insurance charges.

## Objectives
- Perform Exploratory Data Analysis (EDA)
- Clean and preprocess data
- Handle missing values
- Encode categorical variables
- Detect and address heteroscedasticity
- Build and evaluate a regression model

## Techniques Used
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Label Encoding / Factor Conversion
- Linear Regression
- Residual Analysis
- Heteroscedasticity Testing
- Data Visualization

- R
- RStudio
- ggplot2
- dplyr

- lmtest
